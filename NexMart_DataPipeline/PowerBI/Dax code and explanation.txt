
Product Name = 
VAR ShortDesc = TRIM([Short description])
VAR ShortDesc2 = TRIM([Short description 2])
RETURN
    IF(
        NOT(ISBLANK(ShortDesc)) && ShortDesc <> "", 
        ShortDesc,
        IF(
            NOT(ISBLANK(ShortDesc2)) && ShortDesc2 <> "",
            ShortDesc2,
            TRIM([Long description])
        )
    )


Explanation: This DAX code checks if `ShortDesc` has a valid value (not blank or empty), and returns it if true. Otherwise, it checks `ShortDesc2` similarly, and if that's also invalid, returns the trimmed `Long description` as a fallback.


Bad_Field = 
VAR ShortDescBlank = ISBLANK('merged_data_with_completeness_withoutETIM'[Short description])
VAR ShortDesc2Blank = ISBLANK('merged_data_with_completeness_withoutETIM'[Short description 2])
VAR LongDescBlank = ISBLANK('merged_data_with_completeness_withoutETIM'[Long description])
VAR EANBlank = ISBLANK('merged_data_with_completeness_withoutETIM'[EAN])
VAR PictureBlank = 
    ISBLANK('merged_data_with_completeness_withoutETIM'[Picture normal reduced]) || 
    TRIM('merged_data_with_completeness_withoutETIM'[Picture normal reduced]) = ""
VAR TechnicalDetailsBlank = ISBLANK('merged_data_with_completeness_withoutETIM'[Technical details]) || 
    TRIM('merged_data_with_completeness_withoutETIM'[Technical details]) = ""

RETURN
    CONCATENATEX(
        FILTER(
            {
                IF(ShortDescBlank, "Short description", BLANK()),
                IF(ShortDesc2Blank, "Short description 2", BLANK()),
                IF(LongDescBlank, "Long description", BLANK()),
                IF(EANBlank, "EAN", BLANK()),
                IF(PictureBlank, "Picture normal reduced", BLANK()),
                IF(TechnicalDetailsBlank, "Technical details", BLANK())
            },
            NOT(ISBLANK([Value]))
        ),
        [Value],
        ", "
    )
    )

Explanation: This DAX code checks if key fields (Short description, Short description 2, Long description, EAN, Picture normal reduced and Technicals details) are blank or empty, then returns a comma-separated list of missing field names.It helps identify incomplete records by showing which required data points are missing for each row.
